# Foundation Kit

A comprehensive architectural toolkit for Kotlin projects following DDD principles.

## Installation
```kotlin
dependencies {
    implementation("com.rodkrtz:foundation-kit:1.2.0")
}
```

## Publishing credentials

Use one of:
- `~/.gradle/gradle.properties` with `gpr.user` and `gpr.token`
- environment variables `GITHUB_USERNAME` and `GITHUB_TOKEN`
